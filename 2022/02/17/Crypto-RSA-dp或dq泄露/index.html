dp或dq泄露

1.dp和dq的泄露（已知dp，dq，p，q，c）

公式推导转载至https://blog.csdn.net/qq_32350719/article/details/102719279

原本dp和dq的作用是用来加快加解密速度的,但是由于dp和p,dq和q的关系密切,一旦泄漏,将造成很大的安全隐患

这里就不再次赘述了,只写出对解题相关的

  • dp = d mod(p-1)
  • dq = d mod(q-1)
  • InvQ * q = 1 mod p

数学推导最后得出：

  • m1 = c^{dq }mod q
  • m2 = c^{dp} mod p
  • m = (((m2−m1)∗p−1 mod q)p+m1) mod n
